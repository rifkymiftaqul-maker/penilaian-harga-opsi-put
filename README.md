# Penilaian Harga Opsi Put Saham NVIDIA

## Deskripsi
Repository ini berisi implementasi penilaian harga opsi put saham NVIDIA Corporation menggunakan model Black-Scholes dan Long Short-Term Memory (LSTM).

## Metode
- Black-Scholes Model
- Long Short-Term Memory (LSTM)

## Dataset
Data historis saham NVIDIA (NVDA)

## Struktur File
- Pengumpulan_dan_Pembersihan_Data.ipynb
- Statistik_Deskriptif.ipynb
- Perhitungan_Variabel_Turunan.ipynb
- Pembagian_dan_Penskalaan_Data.ipynb
- Implementasi_Mode_Black-Scholes.ipynb
- Pencarian_Hiperparameter_Optimal.ipynb
- Implementasi_Model_LSTM.ipynb
- Hasil_dan_Analisis_Harga_Penilaian.ipynb

## Cara Menjalankan
1. Install library yang dibutuhkan
2. Jalankan notebook secara berurutan
